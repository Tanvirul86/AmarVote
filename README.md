# AmarVote - Election Monitoring & Management System

Secure real-time election monitoring platform for Bangladesh Election Commission.

## Features

- **Landing Page**: Role-based access with three user types (BEC Admin, Presiding Officer, Law Enforcement)
- **Secure Login**: Role selection with demo credentials for testing
- **Role-Based Access Control (RBAC)**: All actions logged and monitored
- **Modern UI**: Clean design with Tailwind CSS

## Tech Stack

- Next.js 14+ (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
